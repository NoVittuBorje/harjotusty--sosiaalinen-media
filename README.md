<h5>Link to site https://frontend-harjotus-sosi.fly.dev/</h5>

<h5>To run this locally.</h5>
<p>You need amazon s3 bucket and mongodb database, you only need the amazon s3 bucket if you want the ability to upload images. And you can make a local mongodb database if you want to test the project</p>
<ol>
  <li>Download project. </li>
  <li>Make .env to backend folder. </li>
  <li>Look at modelenv.txt and copy all neccessary variables to .env file and fill them. </li>
  <li>Go to frontend/src/env.jsx and change deployment to false.</li>
  <li>In terminal run "npm run dev" in both backend and frontend folders.</li>
</ol>
